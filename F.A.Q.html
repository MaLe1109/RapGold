F.A.Q


Willkommen zu den häufig gestellten Fragen!

1. Was ist RapGold?
RapGold ist eine Plattform, auf der sie ihre eigenen Song Texte machen können. wenn sie wollwn können sie ihre Texte mit Breunde teilen

2. Wie kann ich meine Texte teilen?
Sie können ihre Texte auf RapGold teilen, indem sie auf den "Teilen" Button klicken und den Link an ihre Freunde senden.

3. Gibt es eine Möglichkeit, meine Texte zu speichern?
Ja, sie können ihre Texte auf RapGold speichern, indem sie auf den "Speichern" Button klicken. sollten sie vergessen zu speichern und sie die Seite/App einfach verlassen wir ihr geschriebene Text automatisch gespeichert

4. Wie kann ich meine Texte bearbeiten?
Sie können ihre Texte auf RapGold bearbeiten, indem sie auf den "Bearbeiten" Button klicken und ihre Änderungen vornehmen.

5. Gibt es eine Möglichkeit, meine Texte zu löschen?
Ja, sie können ihre Texte auf RapGold löschen, indem sie auf den "Löschen" Button klicken.

6. Wie kann ich meine Texte veröffentlichen?
Unsere Plattform ist nicht so wie andere Plattformen, sie können ihre Texte nicht veröffentlichen. sie können sie mit freunden teilen sie können aber ferifiziert Künstler/in anfragen ob sie sich denn song anschauen

7. wie werde ich ein verifizierter Künstler?
Um ein verifizierter Künstler zu werden, müssen sie ihre Identität nachweisen und ihre musikalischen Fähigkeiten unter Beweis stellen. Nach dem unser zuverlessliches team sie eine weile observierte hat enscheidet under Team ob wir ihnen eine anfrage schicken per Email nachnach können sie sich bewerben

8. Wie kann ich den Support kontaktieren?
Sie können den Support von RapGold kontaktieren, indem sie auf den "Support" Button klicken und das Kontaktformular ausfüllen.

9. Gibt es eine Möglichkeit, meine Texte zu exportieren?
Ja, sie können ihre Texte auf RapGold exportieren, indem sie auf den "Export" Button klicken und das gewünschte Format auswählen.

10. Wie kann ich meine Texte drucken?
Sie können ihre Texte auf RapGold drucken, indem sie auf den "Drucken" Button klicken und die Druckoptionen auswählen.
